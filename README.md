# 古代行军游戏

一个基于HTML5 Canvas的古代行军主题游戏，玩家可以扮演指挥官带领骑兵队伍进行行军。

## 特性

- 多种骑兵阵型（方阵、圆阵、箭阵、横阵）
- 士气系统
- 动态增减士兵
- 流畅的动画效果
- 支持键盘和触摸屏操作

## 如何游玩

### 电脑端操作
- 方向键：控制指挥官移动
- 空格键：发出命令
- F键：切换阵型
- C键：士兵欢呼
- +/-键：增减士兵

### 移动端操作
- 左侧区域：滑动控制移动
- 右下角按钮：
  - 命令：发出命令
  - 阵型：切换阵型
  - 欢呼：士兵欢呼
  - +/-：增减士兵

## 安装说明

1. 克隆仓库：
```bash
git clone [仓库地址]
```

2. 进入项目目录：
```bash
cd [项目目录]
```

3. 使用任意HTTP服务器运行项目，例如：
```bash
python -m http.server 8000
```

4. 在浏览器中访问：
```
http://localhost:8000
```
